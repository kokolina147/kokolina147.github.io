Detailed Project List

  • 🚗 2 DOF Mobile Manipulator Robot — A teleoperated mobile robot equipped with a 2-DOF arm and gripper, controlled via a PS3 controller using ESP32 Bluetooth. Designed with 3D-printed PLA parts and powered by a 3S2P 18650 Li-ion battery system.
  • 🌡️ Environmental Sensing Bot — An IoT-enabled robot for real-time temperature, humidity, and gas detection using the DHT22 sensor and a gas sensor module, integrated into the mobile manipulator platform.
  • 📊 Embedded UI Dashboard — A responsive display interface for visualizing microcontroller data, built using a Cheap Yellow Display and designed for intuitive interaction and real-time system feedback.
  • 🔧 Cyber Truck Mobile Robot — A futuristic mobile robot concept inspired by Tesla’s Cybertruck, controlled using a PS3 controller via Bluetooth, featuring modular hardware architecture and advanced motion control algorithms.
  • 💻 IoT Control System — A Wi-Fi/Bluetooth-enabled embedded control system that integrates sensors and actuators for remote monitoring and automation, leveraging ESP32 and MQTT protocols.
